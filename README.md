# Clínica Dermatológica — Website

Website institucional desenvolvido para uma clínica de dermatologia, com o objetivo de apresentar a clínica os serviços oferecidos os profissionais e facilitar o contato com pacientes.

O projeto foi desenvolvido com foco em design moderno, navegação intuitiva e experiência do usuário.

Visão Geral

Este projeto consiste em uma página web responsiva que apresenta informações sobre a clínica, seus procedimentos dermatológicos e formas de contato.

O site possui um layout elegante e profissional com atribuições exclusivas.

Funcionalidades:  
Página inicial com apresentação da clínica
Sessão de sobre a clínica
Seção sobre a profissional
Seção de procedimentos dermatológicos
Galeria ou carrossel de imagens
Área de contato
Integração com WhatsApp
Design totalmente responsivo
Animações e interações com JavaScript

Tecnologias Utilizadas:  
HTML5
CSS3 
JavaScript
Google Fonts
Design Responsivo (Mobile First)
